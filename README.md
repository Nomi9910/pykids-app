# 🐍 PyKids — Learn Python By Doing

An interactive Python learning app for kids ages 10–14, built with React.

## Features
- 10 modules covering core Python concepts
- 34 tasks with real in-browser code execution
- Animated space-themed background
- Hints, examples, and live output for every task
- Progress tracking across all modules

## Modules
1. 👋 Say Hello! — `print()`
2. 📦 Variables
3. 🤔 If / Else
4. 🔁 For Loops
5. 🔤 String Methods
6. 📝 Lists
7. 🔷 Sets
8. 📖 Dictionaries
9. ⚙️ Functions
10. 🚀 Mini Projects

## Getting Started
```bash
npm install
npm start
```

## Tech Stack
- React 18 (hooks)
- In-browser Python-to-JavaScript transpiler
- HTML5 Canvas animated background
- No external runtime dependencies
